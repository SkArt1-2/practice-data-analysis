# Учебная практика — Анализ и визуализация данных на Python

## Описание проекта

Данный проект выполнен в рамках учебной (ознакомительной) практики по направлению подготовки **38.03.05 «Бизнес-информатика»**.

Целью работы является изучение основных методов обработки, анализа и визуализации данных с использованием языка программирования Python и библиотек NumPy, Pandas и Matplotlib.

В рамках проекта реализована генерация набора случайных данных, вычисление основных статистических характеристик, очистка и преобразование данных, построение графиков, а также формирование структуры DataFrame для дальнейшего анализа.

## Используемые технологии

При разработке проекта использовались следующие технологии и библиотеки:

- Python 3;
- NumPy;
- Pandas;
- Matplotlib;
- Math.

## Возможности программы

Программа выполняет следующие действия:

- генерирует набор из 1000 случайных целых чисел в диапазоне от -10000 до 10000;
- формирует объект Series библиотеки Pandas;
- вычисляет минимальное и максимальное значения;
- рассчитывает сумму элементов;
- вычисляет среднеквадратическое отклонение;
- определяет количество повторяющихся значений;
- выполняет округление данных до сотен;
- строит линейный график исходных данных;
- строит гистограмму распределения данных;
- формирует DataFrame;
- отображает отсортированные по возрастанию и убыванию данные на одном графике.

## Установка

Перед запуском программы необходимо установить требуемые библиотеки.

```bash
pip install numpy pandas matplotlib
```

или

```bash
pip install -r requirements.txt
```

## Запуск программы

Для запуска проекта необходимо выполнить команду:

```bash
python main.py
```

В результате выполнения проекта были получены практические навыки работы с библиотеками Python для анализа данных, освоены методы статистической обработки информации, построения графиков и формирования структурированных наборов данных.
